# Gefährdungsbeurteilung – OCHS Rohrleitungsbau (Web-App / PWA)

Installierbare Web-App der digitalen Gefährdungsbeurteilung. Läuft auf iPhone/iPad, Android und PC.

- **index.html** – Standard-Gefährdungsbeurteilung (für die Baustelle)
- **fix.html** – „fix"-Variante (alle Maßnahmen vorbelegt; jährliche Prüfung GF)
- Über „Zum Home-Bildschirm hinzufügen" wie eine App installierbar; nach dem ersten Laden offline nutzbar.
- Eingaben werden nur lokal auf dem Gerät gespeichert (nichts wird hochgeladen).
